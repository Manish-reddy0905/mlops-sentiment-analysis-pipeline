# End-to-End MLOps Pipeline

## Project Overview
This project demonstrates a complete MLOps pipeline for deploying an NLP sentiment analysis model.

## Features
- PyTorch model
- ONNX optimization
- FastAPI deployment
- Docker containerization
- Google Cloud Run deployment
- GitHub Actions CI/CD

## Technologies Used
- Python
- PyTorch
- ONNX
- FastAPI
- Docker
- Google Cloud
- GitHub Actions

## Workflow
1. Train model
2. Export ONNX
3. Build FastAPI
4. Dockerize app
5. Push to GCR
6. Auto deploy via CI/CD
